# Security Policy

## Supported Versions

The bot is developed using **PHP 8.2** and relies on the **Telegram Bot API**. Below are the supported versions of the project:

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |
| < 1.0   | :x:                |

## Requirements

- **PHP Version**: The project only supports **PHP 8.2** or higher. Vulnerabilities reported on older PHP versions will not be addressed.
- **Telegram API**: All functionalities are dependent on the official Telegram Bot API. Ensure you are using an updated and secure environment.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please follow these steps:

1. **Contact the Maintainer**: Send a message on Telegram to [@ramykatoru](https://t.me/ramykatoru) with a detailed description of the vulnerability, including steps to reproduce it.
2. **Response Time**: You can expect an acknowledgment within **48 hours** of reporting the issue. Updates will be provided regularly until the issue is resolved.
3. **Resolution Process**: If the vulnerability is valid, we will:
   - Investigate and create a fix.
   - Notify you about the resolution.
   - Release an update to address the issue.

Your cooperation in keeping this project secure is highly appreciated!
